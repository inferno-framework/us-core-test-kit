# FHIR NPM Package Supplement Files

The files in this directory will supplement files in the hl7.fhir.<version>.core.tgz package.
Add files to this directory if you need to add information that may be missing from the package.
These files will not supersede the files in the IG.

# Vital Sign Profiles

There are several vital sign profiles StructureDefinition defined in FHIR R4 which are used by US Core v3.1.1:

* StructureDefinition-bodyheight.json
* StructureDefinition-bodytemp.json
* StructureDefinition-bodyweight.json
* StructureDefinition-bp.json
* StructureDefinition-heartrate.json
* StructureDefinition-resprate.json
